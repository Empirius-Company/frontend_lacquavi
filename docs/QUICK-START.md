## 🔥 FAÇA AGORA (30 minutos)

### 1️⃣ Google Search Console
```
1. Acesse: https://search.google.com/search-console
2. Clique "Adicionar propriedade"
3. Digite: lacquaminas.com.br
4. Escolha "Verificação por meta tag"
5. Copie o código: content="XXXXXXXXXXXX"
6. Edite: frontend/index.html ~linha 15
   Atualize: <meta name="google-site-verification" content="COLOQUE_AQUI" />
7. Clique "Verificar" 
8. Depois: Clique "Sitemaps" → Cole: https://lacquaminas.com.br/sitemap.xml
```

### 2️⃣ Google Business Profile
```
1. Acesse: https://www.google.com/business
2. Procure "Lacqua" ou crie nova empresa
3. COMPLETE:
   - Nome: Lacqua Minas Shopping
   - Telefone: +55 31 [seu número]
   - Endereço: Minas Shopping, Belo Horizonte, MG
   - Horários: Seg-Sex 10h-22h, Sábado 10h-23h, Domingo 12h-21h
   - 5+ fotos da loja
   - Website: https://lacquaminas.com.br
4. Pronto! Você aparecerá no Google Maps
```

### 3️⃣ Validar implementação
```bash
cd frontend
node tools/seo-check.js
```
